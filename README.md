# TestCasePUN

Проект демонстрирует простую многопользовательскую сцену на Photon Unity Networking (PUN 2).
Каждый игрок при запуске попадает в общую комнату и управляет своим объектом (Capsule) на плоскости (Plane) с помощью клавиш WASD.

Синхронизация позиций между игроками реализована вручную через
PhotonNetwork.RaiseEvent — без использования RPC, PhotonTransformView или других встроенных синхронизаторов.
